# TestPilotPro

Professional Selenium Automation Framework using Python + Pytest + Page Object Model.

## Features

- Login Automation
- Invalid Login Testing
- Data Driven Testing
- Add To Cart Testing
- Checkout Testing
- Screenshot Capture
- HTML Reports
- Logging
- Explicit Waits
- Config Driven Framework

## Tech Stack

- Python
- Selenium
- Pytest
- WebDriver Manager

## Run Project

pip install -r requirements.txt

pytest -v --html=reports/report.html --self-contained-html
